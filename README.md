# Slack投稿メッセージ抽出ツール

週次報告の投稿メッセージを抽出し、  
テンプレートフォーマットに整形したテキストファイルを投稿するツールです  

## 実行に必要
* Slack API Token(チャンネル履歴取得権限&ファイルアップロード権限)  

## 使用言語およびバージョン
* Java8

## 使用するサービス
* AWS lambda
  * ツールの実行環境
* AWS SSM
  * Token含む機密情報や設定値はこちらに保存

## オプションで使用するサービス
* AWS API Gateway
  * API Gateway経由でlambda関数を実行
